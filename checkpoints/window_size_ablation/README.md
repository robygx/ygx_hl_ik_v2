# Window Size Ablation / 时序窗口大小消融实验

## 目的

验证 Mamba 时序建模对运动平滑度 (Jerk) 的影响。

## 配置

| 目录 | 窗口大小 | 训练日期 | 模型文件 |
|------|----------|----------|----------|
| `W1_window_size1/` | W=1 | 2026-02-27 | `best_model_w1.pth` (8.3MB) |
| `W15_window_size15/` | W=15 | 2026-02-27 | `best_model_w15.pth` |
| `W30_window_size30/` | W=30 | 2026-02-27 | `best_model.pth` |

## 实验结果

| 窗口大小 | Swivel MAE (°) | Jerk | Jerk 降低率 |
|----------|---------------|------|-------------|
| W=30 | 15.87 | 0.304 | ↓ 75% |
| W=15 | 12.14 | 0.240 | ↓ 80% |
| W=1 | 14.41 | 1.207 | 基准 |

## 结论

- **时序建模有效**: Jerk 降低 75-80%
- **W=15 为最优**: 在平滑度和精度间取得最佳平衡
